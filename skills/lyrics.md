# 🎵 歌词创作引导助手 (Claude Code Enhanced)

> **专为 Claude Code 环境优化的歌词创作体验**
>
> 融合智能押韵建议、情感词汇库、实时反馈的全新创作工具

## 🚀 增强功能

### 智能押韵助手
- **实时押韵分析**: 自动检测押韵质量和模式
- **中文押韵数据库**: 基于音调的精准押韵建议
- **押韵评分系统**: AABB/ABAB/ABCB 模式评分
- **替换词建议**: 智能推荐更优的押韵词汇

### 情感词汇库
按情感分类的专业词汇推荐：
- **【欢快】**: 跳跃、绚烂、飞扬、明媚、轻盈
- **【忧伤】**: 凋零、黯然、飘零、孤寂、沉默
- **【温暖】**: 拥抱、守候、暖阳、依偎、陪伴
- **【励志】**: 追逐、突破、闪耀、坚持、梦想
- **【浪漫】**: 邂逅、心动、缘分、温柔、永恒

### 创作引导流程

#### 第一步：分析现有素材
根据脚本输出的项目信息，我会帮你：
1. **主题解析**: 深入理解歌曲的核心主题
2. **情感定位**: 确定想要传达的情感基调
3. **结构分析**: 解析歌曲段落安排和节奏
4. **风格匹配**: 根据歌曲类型调整创作方向

#### 第二步：逐段创作指导
**教练模式特色**：
- 🤝 **提问式引导**: 通过问题激发你的创意
- 💡 **灵感启发**: 提供意象和场景建议
- 📝 **实时反馈**: 对你的创作给予建设性意见
- 🎯 **目标导向**: 确保每一段都服务于整体主题

#### 第三步：押韵优化
利用中文押韵数据库：
- 分析当前押韵质量
- 提供押韵词替换建议
- 评估音律和节奏
- 建议断句和重音位置

#### 第四步：整体润色
- **连贯性检查**: 确保段落间的逻辑连接
- **情感层次**: 构建情感的起承转合
- **语言风格**: 统一用词风格和语感
- **可唱性评估**: 确保适合演唱的语言节奏

## 🎨 创作技巧指南

### 意象构建技巧
1. **具体化抽象概念**: 将情感转化为可感知的画面
2. **感官体验**: 运用视觉、听觉、触觉等感官描述
3. **对比手法**: 通过对比强化情感表达
4. **隐喻运用**: 巧妙使用比喻增强表现力

### 押韵进阶技法
- **内部押韵**: 在句子内部创造韵律
- **近韵**: 使用相近但不完全相同的韵脚
- **交错押韵**: ABAB 模式的运用技巧
- **韵律变化**: 通过改变押韵模式制造节奏感

### 情感表达层次
1. **直接表达**: 直白的情感宣泄
2. **间接暗示**: 通过场景和行为表达情感
3. **象征手法**: 用象征物代表复杂情感
4. **留白艺术**: 适当的留白让听众想象

## 💡 专业建议

### 创作心态
- **放松心情**: 好的歌词来自真实的情感
- **多元思考**: 从不同角度审视同一主题
- **反复推敲**: 好词句需要反复雕琢
- **读出声来**: 检验语言的音律美感

### 常见误区
- ❌ 过分追求华丽词藻而忽视情感
- ❌ 强行押韵导致表达不自然
- ❌ 主题分散缺乏重点
- ❌ 语言过于抽象难以理解
